# Bj35-bot

一个用于管理机器人的Web UI系统
看个乐呵，不一定对

## 项目背景
本项目旨在为Bj35机器人提供一个可视化的管理界面，方便用户配置和监控机器人状态。

## 主要功能
- 机器人状态监控
- 消息发送管理
- 系统配置管理
- 访问令牌管理

## 技术栈
- 前端：HTML/CSS/JavaScript
- 后端：Python Flask
- 数据库：SQLite

## 快速开始
1. 安装依赖：`pip install -r requirements.txt`
2. 启动服务：`python src/python-backend/api.py`
3. 访问 `http://localhost:5000`
